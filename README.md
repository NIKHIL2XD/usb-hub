# USB-HUB v1
A customized 2 port usb hub

## Parts Used

- SL2.1A board : C192893
- 1 USB 2.0 type A male : C319178
- 2 USB 2.0 type A female : C319178
- 1 Shotkey : C48192
- 5 10uF Capacitor : C19702
